# Lead Scan Data

This folder holds the output of the Research Assistant suburb scanner.

- One file per scan: `[suburb]-[YYYY-MM-DD].md` (e.g. `ponsonby-2026-07-14.md`)
- Generated by running `/scan [suburb]` in Claude Code
- Plain Markdown with tables, so the whole folder can be synced or copied
  straight into an Obsidian vault

Each file lists cafés, barbers, salons, and mechanics in the suburb with
name, type, review count, and address. Businesses with under 50 reviews are
flagged with ⚠️. Review counts that couldn't be verified are recorded as
`unknown`, never guessed.
